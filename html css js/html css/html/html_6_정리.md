이미지 표시(img 태그)
---

이미지가 가져다 주는 효과는 사용자의 관심을 끌고, 정보를 직관적으로 전달하며, 브랜드 이미지를 구축하는 핵심 요소

        <img src="이미지경로" alt="이미지설명">

속성
---

src (source) 
----

이미지 파일의 경로를 지정합니다. 

컴퓨터에게 "어디에 있는 이미지를 가져올까?"를 알려주는 주소 같은 역할이에요.

alt (alternative) 
----

이미지를 볼 수 없을 때 대신 보여줄 텍스트입니다. 

시각장애인을 위한 스크린 리더나 이미지 로딩 실패 시에 사용됩니다.
