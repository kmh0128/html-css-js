시맨틱(Semantic)이란
---

시맨틱(Semantic)= 의미론적이란 의미이다.

시맨틱 마크업이란 HTML 태그가 단순히 모양을 만드는 것이 아니라, 콘텐츠의 의미를 명확히 표현하는 것을 뜻한다.

시맨틱 마크업의 장점
----

접근성 향상: 스크린 리더 같은 보조 기술이 콘텐츠를 정확히 해석

SEO 최적화: 검색엔진이 페이지 구조와 내용을 더 잘 이해

코드 가독성: 개발자가 코드의 의미를 쉽게 파악

유지보수성: 구조가 명확해 수정과 관리가 용이
