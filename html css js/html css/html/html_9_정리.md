인용문 태그
---

마치 신문에서 누군가의 발언을 따옴표로 감싸는 것

HTML에서 인용문을 나타내기 위해서는 blockquote 태그를 사용합니다.

인용문은 들여쓰기가 된다.

            q 태그- 짧은 인용문을 문장 안에 포함 (일반적인 따옴표 역할)
            
            code- 짧은 코드를 인라인으로 표시
            
            pre- 긴 코드 블록을 원본 형태로 표시
            
            cite- 인용 출처 표시
