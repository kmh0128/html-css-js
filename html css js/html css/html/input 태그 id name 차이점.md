차이점
---

id 속성 - 고유한 식별을 목적으로 하는 경우 사용( 해당 요소의 고유식별자(ID, unique identifier)를 지정하는 속성)

class 속성 - css 쿼리문 밑 재사용을 목적으로 하는 경우 사용

name 속성 - form 컨트롤 요소의 값(value)을 서버로 전송하기 위해 필요한 속성

id 속성 사용 목적
---

해당 요소를 스타일하기 위한 CSS의 고유한 선택자로 사용

해당 요소를 자바스크립트로 식별하기 위해 사용

해당 요소를 URL의 프래그먼트 요소로 식별하기 위해 사용
