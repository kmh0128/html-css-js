div 태그
---

레이아웃을 만들거나 콘텐츠를  블록 레벨 요소로, 큰 영역을 나누는(division) 컨테이너로 사용하는 태그

방을 칸막이로 나누는 것처럼, 웹페이지의 섹션을 구분할 때 사용

콘텐츠를 컨테이너처럼 블록으로 감싸 스타일이나 속성, 스크립트를 위한 영역으로 나타내기 위해 주로 사용됩니다.


특징
---


      <div> 태그는 순전한 컨테이너로서 본질적으로 어떤 의미도 없다.

      의미가 있는 태그(예를 들어, <article>이나 <nav>)를 사용할 수 없는 상황에서만 <div> 태그를 사용

      <div> 태그에 role 속성을 사용해서 구조나 의미를 부여할 수 있습니다.

      단순 텍스트나 텍스트에 관련된 마크업 등 구문 콘텐츠에 스타일이나 속성의 범위를 적용하기 위해서는 <span> 태그를 사용하는게 좋으며, <div> 태그는 적당하지 않다.
