header 태그란
----

HTML5에서 도입된 시맨틱 태그 중 하나로, 문서나 섹션의 머리말 영역을 의미합니다.

페이지에서 가장 중요한 정보가 담기는 영역이다.

header 태그의 특징
---

의미적 명확성: div 태그와 달리 '머리말'이라는 명확한 의미를 가집니다.

접근성 향상: 스크린 리더 같은 보조 기술이 페이지 구조를 더 잘 이해합니다.

SEO 최적화: 검색 엔진이 페이지의 중요한 정보를 더 쉽게 찾을 수 있습니다.

유연한 사용: 전체 페이지뿐만 아니라 개별 섹션의 헤더로도 사용 가능합니다.
