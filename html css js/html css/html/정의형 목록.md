정의형 목록이란?
---
            
            <dl>: definition list의 약자로, 정의형 목록을 작성시 사용하는 태그.
              
            <dt>: definition term의 약자로, 정의할 용어의 제목을(용어)을  작성할 때 나타냅니다.
              
            <dd>: definition description의 약자로, 용어 제목에 대한 설명을 나타냅니다.
              
dl -> 정의형 목록

dt -> 용어의 제목

dd -> 용어의 설명
