엔티티 코드란?
---

엔티티코드는 HTML에서 특수기호나 다국어를 사용하기 위한 코드를 뜻하며, 모든 엔티티코드는 &으로 시작해서 ; 기호로 끝납니다.

사용하는 이유
---

특수기호나 중국어를 비롯한 외국 문자를 HTML 문서에 직접 삽입하면 깨지는 경우들이 있습니다.

UTF-8로 작성된 웹 문서는 다국어 및 특수 문자를 지원하여 특수문자 및 외국어를 표현할 수 있지만

 EUC-KR으로 HTML 문서가 되어 있는 경우에는 표현이 불가능해서 무조건 문자열이 깨지게 되는경우가 있다.

  엔티티코드를 사용하면  EUC-KR과 같은 2Byte 문자열 셋에도 특수문자나 외국어를 표현할 수 있습니다.
