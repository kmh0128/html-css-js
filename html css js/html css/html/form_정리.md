폼(Form)이란?
----


HTML 에선 브라우저의 내장 요소를 사용하여 웹 양식에 대한 지원을 제공한다.

상호작용이라고도 한다.

상호 작용이란 HTML 요소와 사용자가 서로 상호 작용하여 작동하는 HTML 요소를 의미합니다.

HTML 에선 브라우저의 내장 요소를 사용하여 웹 양식에 대한 지원을 제공한다.

- 입력 칸들: 사용자가 정보를 입력하는 공간
  
- 제출 버튼: 입력한 정보를 서버로 보내는 버튼
  
- 폼 태그: 이 모든 것을 하나로 묶어주는 컨테이너

폼(form)의 속성
---

action: 폼 데이터를 어디로 보낼지 정하는 주소 (목적지)

method: 데이터를 어떤 방식으로 보낼지 정하는 방법 (GET 또는 POST)

input 태그
---

입력을 뜻하는 의미이다.

HTML에서 사용자에게 한 줄만 입력 받는 요소들을 나타낼 때 사용하는 태그이다.

사용자가 정보를 입력하는 부분을 만들어야 할 때 사용

type에 따라 어떤 형식으로 폼 양식을 정할지 결정한다 (버튼 / 이메일 입력 / 비밀번호 입력 ..등)

input 속성
---

상호작용 요소에 함께 사용할 수 있는 속성들

주로 사용되는 속성

readonly: 읽기 전용 필드로 만들기

placeholder: 힌트 표시(필드 클릭시 내용 사라짐)

maxLength: 텍스트 필드에 최대로 입력할 수 있는 문자의 개수 지정

위 속성은  input 태그의 type 속성의 값이 text,email,search,password,tel,url인 경우에만 적용

required: 필수 입력 필드 지정(빈칸이면 안 넘어감)

type: input 태그의 타입을 지정단순히 텍스트입력창인지 이메일, 전화번호 양식 혹은 파일첨부, 동영상 첨부 인지
