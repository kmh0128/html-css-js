화면에 표시될 콘텐츠, <body>
----

실제 브라우저 화면에 표시될 내용을 입력하는 태그

여기에는 다음과 같은 유형의 태그들이 포함될 수 있다.

.텍스트를 표시하는 태그

.이미지를 표시하는 태그

.각종 사용자 인터페이스(버튼, 입력란, 드롭다운 메뉴 등)를 나타내는 태그

 기본 문서 구조를 통해 확인했듯, 태그 안에 태그를 포함하는 방식으로 콘텐츠를 다양하게 구성할 수 있습니다.

속성(attribute)
---

속성(attribute)이 태그에 부가 기능을 추가하기 위해 작성하는 것으로, 선택사항입니다.

 예
 --

        <태그의이름 속성명="속성값">콘텐츠</태그의이름>  


        <태그의이름 속성명="속성값">  


태그에 속성을 추가하는 작업에는 따로 개수 제한이 정해져 있지 않습니다. 

사용자는 하나의 태그에 여러 개의 속성을 필요한 만큼 추가할 수 있습니다

주의할 점
----

태그의 이름이나 각 속성 간의 구분을 정확하게 할 수 있도록 공백 문자(space)를 적절히 추가해 주어야 한다는 점입니다. 

태그 안의 텍스트들이 공백 없이 이어서 쓰여있는 경우 웹 브라우저가 이를 제대로 해석하지 못 할 수도 있습니다.
