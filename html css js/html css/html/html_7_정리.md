strong 태그
---

중요도가 높은 내용

감싸고 있는 콘텐츠를 굵게 표시하는 태그

인라인 요소를 만드는 태그이다.

인라인(inline) 요소
---

자기(컨텐츠)에게 필요한 만큼의 공간만 차지한다.

너비와 높이 설정 불가능

예시 태그
--

            <a>, <span>, <b>, <i>, <img>, <strong>, <em>, <mark> 등

블록(block) 요소
--

한줄 전체를 차지하는 요소

너비(width), 높이(height) 설정가능

예시태그
---

          <div>, <p>, <h1> ~ <h6>, <ul>, <ol>, <li>, <section>, <article>, <nav>등
