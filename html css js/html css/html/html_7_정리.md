---

strong 태그
---

중요도가 높은 내용

감싸고 있는 콘텐츠를 굵게 표시하는 태그

인라인 요소를 만드는 태그이다.

인라인(inline) 요소
---

자기(컨텐츠)에게 필요한 만큼의 공간만 차지한다.

너비와 높이 설정 불가능

요소들이 수평으로 나열된다.

스타일의 경우 margin 또는 padding의 좌우만 적용가능하면서 위아래는 제한적이다.

용도는 텍스트와 관련된 요소나 작은 아이콘등에 사

예시 태그
--

            <a>, <span>, <b>, <i>, <img>, <strong>, <em>, <mark> 등

블록(block) 요소
--

한줄 전체를 차지하는 요소(화면에서 사용가능한 최대 너)

너비(width), 높이(height) 설정가능

요소들이 수직으로 쌓인다.

margin 또는 padding 위 아래 좌 우 모두 적용가능

용도는 레이아웃을 사용할때 주로 사용된다.

예시태그
---

          <div>, <p>, <h1> ~ <h6>, <ul>, <ol>, <li>, <section>, <article>, <nav>등
