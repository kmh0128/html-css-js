테이블의 행과 열
---

HTML 테이블은 행(row)과 열(column)로 구성된 격자 형태

엑셀 스프레드 시트처럼 데이터를 체계적으로 표현하는 방법이다.

숫자 데이터나 비교 정보를 보여줄 때 유용

명확한 구조를 만들어서 스크린 리더나 검색 엔진이 데이터의 관계를 정확히 파악할 수 있게 하기위해 사용

테이블 주요 태그
----

                <table>: 전체 테이블을 감싸는 컨테이너 (집의 골조 같은 역할)
                  
                <thead>: 테이블 머리글 영역 (제목 행들을 묶어줌)
                  
                <tbody>: 테이블 본문 영역 (실제 데이터들이 들어가는 곳)
                  
                <tr> (table row): 테이블의 한 행을 나타냄
                  
                <th> (table header): 제목 셀 (굵게 표시되고 가운데 정렬)
                  
                <td> (table data): 일반 데이터 셀
