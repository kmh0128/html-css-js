font-weight
----

font-weight 속성은 글씨의 굵기(웨이트)를 설정합니다.

            font-weight: normal;
            
            font-weight: bold; 
            
            font-weight: lighter;
            
            font-weight: bolder;


normal
---

일반 글꼴이며, 400 입니다. 숫자는 폰트의 두께를 숫자화한 수치 100~900까지의 단위

bold
---

굵은 글꼴입니다. 700 입니다.

bolder
---

부모 요소보다 두꺼운 가중치의 두께

lighter
---

부모 요소보다 얇은 가중치의 두께

number
---

글꼴의 굵기를 100~900까지 100단위로 끊습니다.
