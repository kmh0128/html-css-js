CSS 개념
--

CSS(Cascading Style Sheets)는 웹페이지의 시각적 표현을 담당하는 언어입니다. 

HTML이 웹페이지의 구조와 내용을 정의한다면, CSS는 그 내용이 어떻게 보여질지를 결정합니다.

HTML은 집의 기둥, 벽, 문, 창문 같은 기본 구조

CSS는 페인트를 칠하고, 벽지를 도배, 가구를 배치하고, 조명을 설치하는 것 즉 CSS는 웹문서를 꾸며주기 위해 사용하는 언어

특징
---


분리의 원칙: HTML은 내용(구조)에, CSS는 표현(디자인)에 집중

재사용성: 하나의 CSS 파일로 여러 HTML 페이지의 스타일 관리 가능

유지보수성: 디자인 변경 시 CSS만 수정하면 전체 사이트 스타일 변경 가능

Cascading: 여러 스타일 규칙이 적용될 때의 우선순위 시스템

덧붙이기
---

CSS 코드는 CSS 파일에 작성한다. 확장자는 *.css 

CSS 코드는 HTML 문서에 스타일을 추가(정의)하는 언어다.

HTML 문서가 존재하지 않을 때는 단순 텍스트라고 보면 된다.
