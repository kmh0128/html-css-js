font-size 속
---

    font-size: 16px;
    
px(픽셀): 크기단위이면서 동시에 표준적이 단위이다.

표시되는 크기가 고정적이다.

데스크탑이나 모바일 등등 디바이스 종류에 상관없이 동일한 크기이다.

요약:  화면을 구성하는 단위인 화소(픽셀)를 뜻하는 절대 단위입니다.

1px은 화소 1개 크기를 의미

em
--

부모태그의 폰트크기에 영향을 받는 상대적인 크기

부모 요소의 글자 크기를 배수로 계산해 적용하는 상대 단위입니다.

rem
--    

em과의 차이점은 최상위 태그인 HTML 태그의 폰트크기에만 영향을 받는다.

루트 요소의 글자 크기를 배수로 계산해 적용하는 상대 단위입니다.

pt
---

포인트라고 하며, 일반 문서에서 많이 사용하는 절대 단위입니다.


%
--

부모 요소의 글자 크기를 백분율로 계산해 적용하는 상대 단위입니다.
